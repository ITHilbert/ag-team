# Instruktion für Architekt

**Status:** Anforderungen liegen in `ag-team/projekt/temp_anforderungen.md` vor.
**Ziel:** Initialisierung der notwendigen Worker für das MVP (Web-App).

**Deine Aufgaben:**
1.  Analysiere die Anforderungen in `ag-team/projekt/temp_anforderungen.md`.
2.  Entscheide, welche Worker-Rollen benötigt werden (z.B. Frontend-Dev, Backend-Dev, Fullstack, etc.).
3.  Nutze Dein Tool `init_worker(name, role_file)`, um diese Agenten zu erstellen.
    *   Erstelle für jeden Worker eine neue Rollendatei im Ordner `ag-team/agenten/` (z.B. `05_worker_frontend.md`).
    *   Der Inhalt der Rollendatei soll die spezifischen Aufgaben des Workers beschreiben (basierend auf den Anforderungen).
4.  Melde Vollzug an den Moderator via `contact_moderator`.

Bitte führe dies jetzt aus.
